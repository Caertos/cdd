Guía rápida

  1. Instalar: npm install -g cdd-cli o usar el binario.
  2. Iniciar: Ejecuta cdd para ver el panel interactivo.
  3. Listar contenedores: cdd list muestra contenedores locales.
  4. Ver logs: cdd logs <id> para seguir los logs en tiempo real.

Ejemplo rápido

# listar contenedores
cdd list

# ver logs en tiempo real
cdd logs 4a7b3c2d

Si tienes problemas con permisos Docker, asegúrate de que tu usuario esté en el grupo docker o ejecuta con sudo según tu sistema.
