CDD — CLI Docker Dashboard

Command-line dashboard to inspect and manage Docker containers.

View API Documentation Open README on GitHub

Quickstart

  1. Install: npm install -g cdd-cli
  2. List: cdd list
  3. Logs: cdd logs <container-id>

Example

# list containers
cdd list

# follow logs live
cdd logs 4a7b3c2d

If you run into Docker permission issues, add your user to the docker group or run commands with sudo.
